Данный проект представляет собой автономного AI-агента для управления веб-браузером и выполнения сложных многошаговых задач. Агент написан на Python с использованием Playwright для автоматизации браузера и Google Gemini API 

Возможности агента

Агент умеет автономно выполнять задачи в браузере: открывать сайты, анализировать страницы, находить элементы, кликать, вводить текст, скроллить, заполнять формы. Он сам определяет последовательность действий на основе анализа текущего состояния страницы без предзаготовленных селекторов или шаблонов.
Быстрый старт

Клонируйте репозиторий:

git clone https://github.com/yakubka/autonomous_web_agent.git
cd autonomous_web_agent

Создайте виртуальное окружение и установите зависимости:

python -m venv venv

source venv/bin/activate

pip install -r requirements.txt

playwright install

Настройте API ключ в файле .env:

GEMINI_API_KEY=ваш_ключ_от_google_ai_studio

GEMINI_MODEL=gemini-2.0-flash

Запустите агента:

python main.py

Как делать запросы веб-агенту

После запуска программы командой python main.py откроется интерактивная консоль. Вы увидите приглашение "Введите команду или задачу:". 
В этом режиме есть два способа взаимодействия

Просто введите задачу текстом

Найди рецепт пасты

Используйте команды

Доступны следующие команды (начинаются с /):

/task [задача] - выполнить конкретную задачу
/task Найди курсы по Python
/url [адрес] - перейти по указанному URL
/url https://google.com

/status - показать статистику выполненных действий

/stop - остановить текущее выполнение задачи

/exit - выйти из программы
